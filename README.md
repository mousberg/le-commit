# Le Commit

A clean, modern hackathon project setup with Next.js + Tailwind + shadcn/ui.

## Quick Start

### npm
```bash
cd frontend
npm install
npm run dev
```

### bun
```bash
cd frontend
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000)
